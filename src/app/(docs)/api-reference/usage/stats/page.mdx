# Usage Stats

<ApiEndpointBadge method="GET" path="/api/v1/usage" />

Returns the current configuration and lifetime request count for the calling API key.

**Required scope:** none — any valid, non-revoked API key can call this regardless of its scopes.

<Note>
This is a snapshot of the key itself, not a time-series — there's no `days` parameter and no
per-day breakdown. For message/session counts over a time window, use **Analytics Summary** instead.
</Note>

## Response

<ResponseField name="key_prefix" type="string">The non-secret prefix of the API key (for identifying it in a list of keys).</ResponseField>
<ResponseField name="scopes" type="array">The scopes granted to this key, e.g. `["chat", "read"]`.</ResponseField>
<ResponseField name="allowed_ips" type="array">IP allowlist for this key, or `null` if unrestricted.</ResponseField>
<ResponseField name="request_count" type="integer">Lifetime total requests made with this key.</ResponseField>
<ResponseField name="last_used_at" type="string">ISO 8601 timestamp of the most recent request, or `null` if never used.</ResponseField>
<ResponseField name="created_at" type="string">When the key was created.</ResponseField>
<ResponseField name="rate_limit_per_min" type="integer">The per-key rate limit currently in effect (see [Rate Limits](/rate-limits)).</ResponseField>

## Example

```bash title="cURL"
curl "https://api.personaliai.com/api/v1/usage" \
  -H "Authorization: Bearer chatty_sk_your_key"
```

**200 OK:**
```json
{
  "key_prefix": "chatty_sk_7f3a",
  "scopes": ["chat", "read"],
  "allowed_ips": null,
  "request_count": 4821,
  "last_used_at": "2026-07-07T14:32:10Z",
  "created_at": "2026-05-01T09:00:00Z",
  "rate_limit_per_min": 60
}
```
