# List Knowledge Sources

<ApiEndpointBadge method="GET" path="/api/v1/knowledge" />

Returns all knowledge sources (text snippets, crawled URLs, uploaded documents) that the bot uses when answering questions.

**Required scope:** `read`

## Query parameters

<ParamField query="limit" type="integer" default="50">Number of sources to return. Max 200.</ParamField>
<ParamField query="offset" type="integer" default="0">Zero-based offset for pagination.</ParamField>

## Response

<ResponseField name="sources" type="array">
  Array of knowledge source objects.
  <Expandable title="Source object fields">
    <ResponseField name="id" type="string">UUID of the source.</ResponseField>
    <ResponseField name="bot_id" type="string">Bot this source belongs to.</ResponseField>
    <ResponseField name="type" type="string">`"text"`, `"url"`, or `"document"`.</ResponseField>
    <ResponseField name="title" type="string">Display title or URL.</ResponseField>
    <ResponseField name="content_preview" type="string">First 200 characters of the content.</ResponseField>
    <ResponseField name="status" type="string">`"ready"`, `"processing"`, or `"error"`.</ResponseField>
    <ResponseField name="created_at" type="string">ISO 8601 timestamp.</ResponseField>
  </Expandable>
</ResponseField>
<ResponseField name="total" type="integer">Total number of sources for this bot.</ResponseField>

## Example

```bash title="cURL"
curl "https://api.personaliai.com/api/v1/knowledge" \
  -H "Authorization: Bearer chatty_sk_your_key"
```
