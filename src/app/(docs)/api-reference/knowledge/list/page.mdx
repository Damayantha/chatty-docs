# List Knowledge Sources

<ApiEndpointBadge method="GET" path="/api/v1/knowledge" />

Returns all knowledge sources (text snippets and crawled URLs) that the bot uses when answering questions.

**Required scope:** `read`

<Note>This endpoint takes no query parameters — it always returns the full list, newest first.</Note>

## Response

<ResponseField name="sources" type="array">
  Array of knowledge source objects.
  <Expandable title="Source object fields">
    <ResponseField name="id" type="string">UUID of the source.</ResponseField>
    <ResponseField name="type" type="string">`"text"` or `"url"`.</ResponseField>
    <ResponseField name="name" type="string">Display name (for text sources) or the crawled URL.</ResponseField>
    <ResponseField name="status" type="string">Always `"trained"` — sources are indexed synchronously when added.</ResponseField>
    <ResponseField name="char_count" type="integer">Length of the indexed content, in characters.</ResponseField>
    <ResponseField name="crawl_schedule" type="string">For URL sources: `"off"`, `"daily"`, `"weekly"`, or `"monthly"`.</ResponseField>
    <ResponseField name="next_crawl_at" type="string">ISO 8601 timestamp of the next scheduled re-crawl, if any.</ResponseField>
    <ResponseField name="last_crawled_at" type="string">ISO 8601 timestamp of the last crawl, if any.</ResponseField>
    <ResponseField name="created_at" type="string">ISO 8601 timestamp.</ResponseField>
  </Expandable>
</ResponseField>

## Example

```bash title="cURL"
curl "https://api.personaliai.com/api/v1/knowledge" \
  -H "Authorization: Bearer chatty_sk_your_key"
```
