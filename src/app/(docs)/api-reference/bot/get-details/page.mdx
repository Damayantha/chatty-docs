# Get Bot Details

<ApiEndpointBadge method="GET" path="/api/v1/bot" />

Returns configuration details about the bot tied to this API key. Useful for rendering the bot's name, colors, and feature flags in your custom UI.

**Required scope:** `read`

## Response

<ResponseField name="id" type="string">Unique bot identifier (UUID).</ResponseField>
<ResponseField name="name" type="string">Display name of the bot.</ResponseField>
<ResponseField name="welcome_message" type="string">First message the bot sends when a conversation starts.</ResponseField>
<ResponseField name="primary_color" type="string">Hex color for the bot's widget UI (e.g. `"#f97316"`).</ResponseField>
<ResponseField name="selected_model" type="string">AI model: `"gemini"`, `"openai"`, `"anthropic"`, or `"openrouter"`.</ResponseField>
<ResponseField name="lead_capture_enabled" type="boolean">Whether the bot captures visitor contact details.</ResponseField>
<ResponseField name="lead_fields" type="array">Lead fields collected (e.g. `["name", "email", "phone"]`).</ResponseField>
<ResponseField name="calendar_scheduling_enabled" type="boolean">Whether the bot can book meetings.</ResponseField>
<ResponseField name="meeting_provider" type="string">`"google"`, `"calendly"`, or `null`.</ResponseField>
<ResponseField name="response_language" type="string">Forced reply language code, or `""` for auto-mirror.</ResponseField>
<ResponseField name="created_at" type="string">ISO 8601 creation timestamp.</ResponseField>

## Example

```bash title="cURL"
curl https://api.personaliai.com/api/v1/bot \
  -H "Authorization: Bearer chatty_sk_your_key"
```

**200 OK:**
```json
{
  "id": "a1b2c3d4-e5f6-7890-abcd-ef1234567890",
  "name": "Aria — Support Assistant",
  "welcome_message": "Hi! I'm Aria. How can I help you today?",
  "primary_color": "#f97316",
  "selected_model": "gemini",
  "lead_capture_enabled": true,
  "lead_fields": ["name", "email"],
  "calendar_scheduling_enabled": false,
  "meeting_provider": null,
  "response_language": "",
  "created_at": "2026-03-15T10:30:00Z"
}
```
