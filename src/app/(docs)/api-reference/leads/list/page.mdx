# List Leads

<ApiEndpointBadge method="GET" path="/api/v1/leads" />

Returns visitor leads captured through the bot's lead collection flow — typically name, email, phone, and custom fields.

**Required scope:** `read`

## Query parameters

<ParamField query="limit" type="integer" default="50">Number of leads to return. Min 1, max 200.</ParamField>
<ParamField query="offset" type="integer" default="0">Zero-based offset for pagination.</ParamField>

## Response

<ResponseField name="leads" type="array">
  Array of lead objects.
  <Expandable title="Lead object fields">
    <ResponseField name="id" type="string">UUID of the lead.</ResponseField>
    <ResponseField name="bot_id" type="string">Bot that captured this lead.</ResponseField>
    <ResponseField name="session_id" type="string">Conversation session where the lead was captured.</ResponseField>
    <ResponseField name="name" type="string">Visitor's name (if collected).</ResponseField>
    <ResponseField name="email" type="string">Visitor's email (if collected).</ResponseField>
    <ResponseField name="phone" type="string">Visitor's phone (if collected).</ResponseField>
    <ResponseField name="extra" type="object">Any additional collected fields.</ResponseField>
    <ResponseField name="created_at" type="string">ISO 8601 timestamp.</ResponseField>
  </Expandable>
</ResponseField>
<ResponseField name="total" type="integer">Total leads for this bot across all pages.</ResponseField>
<ResponseField name="limit" type="integer">Limit used in this request.</ResponseField>
<ResponseField name="offset" type="integer">Offset used in this request.</ResponseField>

## Example

```bash title="cURL"
curl "https://api.personaliai.com/api/v1/leads?limit=20" \
  -H "Authorization: Bearer chatty_sk_your_key"
```

```python title="Python — paginate all"
import httpx
client = httpx.Client(base_url="https://api.personaliai.com",
                      headers={"Authorization": "Bearer chatty_sk_your_key"})
all_leads, offset = [], 0
while True:
    data = client.get("/api/v1/leads", params={"limit": 100, "offset": offset}).json()
    all_leads.extend(data["leads"])
    if len(all_leads) >= data["total"]: break
    offset += 100
```
