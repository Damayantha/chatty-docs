# List Conversations

<ApiEndpointBadge method="GET" path="/api/v1/conversations" />

Returns messages across all conversation sessions, newest first. Use `GET /api/v1/conversations/{session_id}` to fetch a single thread.

**Required scope:** `read`

## Query parameters

<ParamField query="limit" type="integer" default="50">Number of messages. Min 1, max 200.</ParamField>
<ParamField query="offset" type="integer" default="0">Zero-based offset.</ParamField>

## Response

<ResponseField name="messages" type="array">
  Array of messages with `id`, `session_id`, `role` (`"user"` or `"assistant"`), `content`, `feedback_rating`, `created_at`.
</ResponseField>
<ResponseField name="limit" type="integer">Limit used.</ResponseField>
<ResponseField name="offset" type="integer">Offset used.</ResponseField>

## Example

```bash title="cURL"
curl "https://personaliai-api-376030619262.us-central1.run.app/api/v1/conversations?limit=50" \
  -H "Authorization: Bearer chatty_sk_your_key"
```

**200 OK:**
```json
{
  "messages": [
    {
      "id": "msg-uuid-001",
      "session_id": "visitor-xyz-789",
      "role": "assistant",
      "content": "You're welcome! Is there anything else I can help with?",
      "feedback_rating": "up",
      "created_at": "2026-07-06T14:25:00Z"
    }
  ],
  "limit": 50,
  "offset": 0
}
```
